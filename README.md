# TANZANIA DEMOCRACY SYSTEM - COMPLETE DOCUMENTATION

## SYSTEM OVERVIEW
This is a complete website explaining Democracy in Tanzania and all 6 Presidents of Tanzania since independence. It is built in Swahili language to educate citizens about the country's democratic history.

---

## FILE STRUCTURE (9 Files)

| File Name | Purpose |
|-----------|---------|
| index.html | Homepage - shows democracy info and all presidents |
| styles.css | Controls colors, layout, and appearance |
| script.js | Controls dropdown menu and interactions |
| nyerere.html | Full details page for Julius Nyerere |
| mwinyi.html | Full details page for Ali Hassan Mwinyi |
| mkapa.html | Full details page for Benjamin Mkapa |
| kikwete.html | Full details page for Jakaya Kikwete |
| magufuli.html | Full details page for John Magufuli |
| samia.html | Full details page for Samia Suluhu Hassan |

---

## SYSTEM FUNCTIONALITIES

### 1. Homepage (index.html)
- Democracy section explaining Tanzania's multi-party system, free elections, constitution, and human rights
- Presidents grid showing 6 cards with photos, names, leadership years, and short biography
- Navigation menu with dropdown listing all presidents

### 2. President Pages (nyerere.html, mwinyi.html, etc.)
Each president page contains:
- Full biography - life story and leadership history
- Democracy contribution - how the president strengthened democracy
- Major achievements - list of key accomplishments during their rule
- President photo
- Back link to return home

### 3. Interactive System (JavaScript)
- Dropdown menu - clicking "Marais" shows list of all presidents
- Click photo - clicking any president photo navigates to full details page
- Open and close - dropdown opens and closes properly

- Tanzania flag colors - Green, Blue, Yellow, and Black used in headers
- Responsive design - Works on phones, desktops, and tablets
- Image hover effects - Photos enlarge slightly when mouse hovers
- Card hover effects - Cards rise with shadow when mouse hovers

## CONTENT OVERVIEW

### Democracy in Tanzania:
- Multi-party system since 1992
- Free elections every five years
- Constitution limits leaders' powers
- Human rights and media freedom
- Existing challenges and solutions

### Six Presidents in the System:

| No. | President | Period | Main Contribution |
|-----|-----------|--------|-------------------|
| 1 | Julius Nyerere | 1964-1985 | Father of Nation, Ujamaa, national unity |
| 2 | Ali Hassan Mwinyi | 1985-1995 | Economic reforms, opened multi-party system |
| 3 | Benjamin Mkapa | 1995-2005 | Anti-corruption fight, economic strengthening |
| 4 | Jakaya Kikwete | 2005-2015 | Infrastructure, peace, secondary education |
| 5 | John Magufuli | 2015-2021 | Major projects (SGR, roads), anti-corruption |
| 6 | Samia Suluhu | 2021-present | First female president, transparency, diplomacy |

## SYSTEM REQUIREMENTS

- Storage space ~1MB only

### No Requirements for:
- Database
- Backend server
- API keys
- Login account

## HOW THE SYSTEM WORKS
1. User opens index.html in browser
2. User reads democracy info and views president cards
3. User clicks photo of desired president
4. System navigates to that president's dedicated page
5. User reads full details (biography, democracy contribution, achievements)
6. User returns home by clicking "Back to Home"
7. User uses dropdown menu to browse other presidents easily
   
## SECURITY AND DEPENDENCIES
- No data stored - stateless system
- No external dependencies - all CSS and JS local
- Images from Wikimedia - reliable sources
- Fallback images - appear if main image fails

## SYSTEM OBJECTIVES
1. Educate Tanzanians about democracy and their rights
2. Preserve history of Tanzanian presidents
3. Provide resource for students, researchers, and visitors
4. Promote Swahili language in technology
5. Simplify access to historical information
## SUMMARY
This system is a complete, standalone website explaining Democracy in Tanzania and 6 Tanzanian Presidents in Swahili language. It has separate pages for each president, click-photo interaction to open full details, dropdown navigation menu, and responsive design for all devices. It does not require internet for structure (only for photos) and needs no database.
